# Backend (VideoGamesCatalogue)

1. This is the backend of the "Video Games Catalogue" project. It uses VS 2022 and has unit tests included as well.
2. Before executing this application, make sure you have SQL Server (Express) installed and the below commands are executed in the repo location via command prompt:
  dotnet ef migrations add InitialCreate
  dotnet ef database update
